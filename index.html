Four tools, one suite

Everything your media library needs.

Each tool is a standalone Python script. Use them individually or via the unified HTML dashboard.

📎
Video Merger
Combine multiple video or audio files into a single seamless output.
  • Drag-and-drop file ordering
  • Sort by name, date, or size
  • Stream copy or re-encode
  • Supports mkv, mp4, mp3, flac + more
✨
AI Upscaler
Upscale video and images with Real-ESRGAN — all local, all offline.
  • 480p → 1080p (real content)
  • 720p → 1080p (anime mode)
  • 1080p → 4K (real + anime)
  • Bulk folder upscaling
🏷️
AI File Renamer
Rename messy media files to clean Jellyfin/Plex-ready format automatically.
  • Detects series vs movie
  • Fetches episode titles + air dates
  • Strips junk (x265, BluRay, YTS…)
  • Auto-sorts into season folders
🔊
Default Audio Changer
Fix default audio and subtitle tracks in MKV files without re-encoding.
  • Auto English mode
  • Groups tracks by language
  • Sorts into /NO ENG folders
  • Bulk folder processing
🔒
100% Local
No files ever leave your machine. All AI runs via Ollama or local binaries.
📊
Live Dashboard
HTML dashboard with real-time progress, analytics, and history — no server needed.
⚡
Free APIs
Episode data from TVmaze — no API key required. Optional TMDB for movies.
🗂️
Analytics & History
Tracks every file processed, project usage stats, and combined sizes over time.
